# RenPy-RPA-Auto-Unpacker 🚀

> 牛逼吊炸天的RenPy超级万能RPA解包工具 - 智能解密，一键解包

## 📋 项目简介

**unpacker.py** 是一个RenPy游戏RPA文件自动解包工具。它的最大特色是**自动识别并模拟RenPy游戏的解密逻辑**，能够智能解析各种RPA格式，无需手动配置解密参数，真正实现了一键式全自动解包！

## ✨ 核心特色

### 🎯 最牛逼的点：自动智能解密
本工具的**最大优势**在于能够自动识别和模拟RenPy游戏的解密机制：

- **智能解密逻辑识别**：自动分析RPA文件的加密方式和解密算法
- **无需人工干预**：不需要手动指定解密密钥或算法参数
- **自适应解析**：自动适应不同RenPy版本的游戏加密方式
- **全自动处理**：从检测到解包全程无需用户配置

### 🚀 其他强大功能
- **全自动检测**：自动扫描游戏目录中的RPA文件
- **零配置操作**：只需运行一次即可完成解包
- **完整提取**：保留原始目录结构和所有文件
- **错误处理**：完善的异常处理和进度显示
- **开源自由**：采用GNU GPL v3.0许可证，自由使用和修改

## 📦 快速使用指南

### 环境要求
- Python 3.6 或更高版本

### 安装步骤
1. 下载 `unpacker.py` 文件
2. 将文件复制到游戏根目录（与游戏exe同目录）

### 使用方法
1. 将 `unpacker.py` 复制到游戏根目录：
   ```
   游戏根目录/
   ├── 游戏.exe              # 游戏主程序
   ├── game/                # 游戏数据目录
   │   ├── *.rpa           # RPA资源文件
   │   └── ...
   ├── unpacker.py          # 本解包工具
   └── ...
   ```

2. 运行解包工具：
   ```bash
   python unpacker.py
   ```

3. 等待自动解密和解包完成

### 输出结果
解包后的文件将保存在 `extracted_files/` 目录中，保持完整的原始目录结构。

## 🔧 技术原理

### 智能解密机制
本工具的核心技术在于**自动分析RenPy游戏的解密逻辑**：

1. **动态加载器分析**：自动提取RenPy游戏加载器的解密代码
2. **加密算法识别**：智能识别RPA文件使用的加密算法
3. **密钥自动推导**：无需用户输入即可推导解密密钥
4. **格式自适应**：自动适应不同版本的RPA格式

### 处理流程
1. **自动检测**：扫描`game/`目录下的`.rpa`文件
2. **解密逻辑分析**：自动识别游戏的解密机制
3. **索引解析**：读取RPA文件的内部索引结构
4. **文件提取**：逐个提取并保存解密后的文件
5. **进度显示**：实时显示解包进度和状态

## ❓ 常见问题

### Q: 为什么把工具放在游戏根目录？
A: 工具需要读取游戏目录结构并自动识别游戏的解密逻辑，必须与游戏exe同目录。

### Q: 提示"没有找到任何存档索引"
A: 可能的原因：
   - RPA文件不在`game/`目录中
   - RPA文件使用了特殊的加密方式
   - 游戏目录结构异常

### Q: 解密失败怎么办？
A: 本工具会自动尝试多种解密方式，如果失败可能是因为：
   - 游戏使用了非标准的加密方式
   - RPA文件已损坏
   - 需要更新工具版本

### Q: 支持哪些RenPy版本？
A: 支持大多数基于RPAv3格式加密资源的RenPy游戏，包括使用标准加密的游戏。
